# DS Practices

Data science projects built while practicing Machine Learning.

## Projects

### 1. Credit Card Fraud Detection
- Dataset: Kaggle Credit Card Fraud Detection (284,807 transactions)
- Model: RandomForestClassifier
- Key challenge: Only 0.17% of transactions are fraud
- Result: 74% fraud recall, 97% precision

### 2. Titanic Survival Prediction
- Dataset: Kaggle Titanic Dataset (891 passengers)
- Model: LogisticRegression
- Result: 80% accuracy, 78% recall on non-survivors
